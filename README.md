# Uter-Us 🩸

Your personal, secure, and simple cycle tracker. 

## Features
* **Cycle Tracking:** Log start dates, end dates, symptoms, and personal notes.
* **Privacy First:** Secure authentication ensures your health data is entirely your own.
* **Tech Stack:** Built with Laravel, Tailwind CSS, and Blade.

## Local Setup Instructions
1. Clone the repository.
2. Install PHP dependencies: `composer install`
3. Install Node dependencies: `npm install`
4. Copy the `.env.example` file to `.env` and configure your database (e.g., SQLite or MySQL).
5. Generate an app key: `php artisan key:generate`
6. Run the database migrations: `php artisan migrate`
7. Compile frontend assets: `npm run dev`
8. Start the local server: `php artisan serve`
